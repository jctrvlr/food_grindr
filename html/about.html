Food Grindr

Food Grindr is a web appliction that gathers location of user, and outputs nearby restaurants' information such as name, location, type of cuisine, hours of operation, phone number, and if it delivers. The user also has the option of reviewing the restaurant and to see other user's ratings. It uses the Zomato API and pull info from there.
